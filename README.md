## Currently Building — Fynex

> A next-gen fintech platform redefining how people interact with money. MVP shipping in **~30 days**.

**My contribution — Hybrid Cache Architecture:**

Designed a **Redis × PostgreSQL hybrid caching system** with configurable TTL per API call type — so external calls are made only when strictly necessary. Latency down. Costs down. Reliability up.

```
External API call
      │
      ▼
 [Redis Cache] ──hit──▶ return instantly
      │ miss
      ▼
 [PostgreSQL]  ──hit──▶ repopulate Redis → return
      │ miss
      ▼
 [Live API]    ──────▶ store in Redis + DB → return
```

*Stack: NestJS · PostgreSQL · Redis · TypeScript · Docker*
